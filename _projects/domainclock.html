Domain Clock

Domain Clock generates domain names of the current time. For example, fivethirtyeight.com appears when the time is 5:38 PM.

I created Domain Clock after browsing a web site made of four numbers. I realized that the numbers represent time with digits for hours and minutes.

Visit Domain Clock >

Implemented in PHP, HTML, CSS, and Javascript. Used the jQuery Javascript library. The font for the body is Helvetica. The layout is built on top of Blueprint CSS.
