Domain Clock

Domain Clock tells time using domain names. For example, at 5:38pm, Domain Clock might show fivethirtyeight.com.

I created Domain Clock after browsing a web site made of four numbers and realizing that the numbers represent time with digits for hours and minutes.

Visit Domain Clock >

Implemented in PHP, HTML, CSS, and Javascript. Used the jQuery Javascript library. The font for the body is Helvetica. The layout is built on top of Blueprint CSS.
