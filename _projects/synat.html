I came across the beautiful infographic poster So You Need a Typeface that was designed by Julian Hansen. I thought it might be cool to make a version that is interactive, so I made this website.

Visit the website >

View code on GitHub >

The website starts with the list of all the fonts from the poster. The first thing the user does is choose the kind of project the font is going to be used for, e.g., a logo, an infographic, etc.

The website continues to ask questions until the fonts are filtered to a single font.
